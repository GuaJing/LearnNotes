:sparkles:用户名 @mentions:sparkles:
键入 @ 符号，后面跟一个用户名，将通知该用户来查看评论。这被称之为“@memtion”（@提及），因为你提到了这个人。你也可以 @memtion 组织内部的一个小组。

:sparkles:自动链接网址:sparkles:
任何网址（如 http://www.github.com）都会自动转换为可点击的链接。

:sparkles:删除线:sparkles:
